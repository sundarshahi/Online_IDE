Your Developer API key: {{key}}


GET request:

With this API, you can obtain a user's all submitted codes in json format. You just need to send a valid GET request to the following URL. 

"http://www.mohit-onlineide.herokuapp.com/api"
Include all the following GET parameters when you request the URL:
  1. key -this is your API Developer Key
For obtaining a particular code, include following optional parameter:
  1. title -title of the code you want


GET request Example:

Following code is in Python3:

import requests
url="http://www.mohit-onlineide.herokuapp.com/api"
params={
    "key":"Your_Developer_API_Key",
    "title":"title_of_submitted_code"(optional)
}
r=requests.get(url,params=params)
print(r.content)

POST request:

Creating a new code via our API is very easy. You simply have to send a valid POST request to the URL shown below. Please make sure you are sending the data as the UTF-8 charset! 

"http://www.mohit-onlineide.herokuapp.com/api"
Include all the following POST parameters when you request the URL:
  1. key- this is your API Developer Key
  2. title- Title of new code
  3. code- Code to compile and run
  4. lang- Language code
  5. input- Testcase
Click here for language codes of different languages.

Few points to remember while sending a request:

  • Your title must be unique i.e. there should be no other submission of same title with your key or userid.
  • Title can only contain alphabets, underscore and integer. No special symbol is allowed.
  • Language Code is an integer.


POST request Example:

Following code is in Python3: 

import requests
url="http://www.mohit-onlineide.herokuapp.com/api"
data={
    "key":"Your_Developer_API_Key",
    "title":"Hello_world",
    "code":"print("Hello world")",
    "lang":5,
    "input":" "
    ""
}
r=requests.post(url,data=data)
print(r.content)
Click here for language codes of different languages.


PATCH request:

With this API you can update your submitted code. You will need to send a valid PATCH request to the URL below to update the data: 

"http://www.mohit-onlineide.herokuapp.com/api"
Include all the following PATCH parameters when you request the URL:
  1. key- this is your API Developer Key
  2. title1- Title of existing code i.e. code to be updated.
  3. title- Title of new code
  4. code- Code to compile and run
  5. lang- Language code
  6. input- Testcase
Click here for language codes of different languages.
Please read here, all the points to remember while sending a PATCH request.


PATCH request Example:

Following code is in Python3: 

import requests
url="http://www.mohit-onlineide.herokuapp.com/api"
data={
    "key":"Your_Developer_API_Key",
    "title1":"Hello_world",
    "title":"Hello_earth",
    "code":"print("Hello earth")",
    "lang":5,
    "input":" "
    ""
}
r=requests.patch(url,data=data)
print(r.content)

DELETE request:

With this API you can delete your submitted code. You will need to send a valid DELETE request to the URL below to delete the data: 

"http://www.mohit-onlineide.herokuapp.com/api"
Include all the following DELETE parameters when you request the URL:
  1. key- this is your API Developer Key
  2. title- Title of existing code i.e. code to be deleted.


DELETE request Example:

Following code is in Python3: 

import requests

data={
    "key":"Your_Developer_API_Key",
    "title":"Hello_earth",
    ""
}
